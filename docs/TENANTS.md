## Cómo agregar un nuevo proveedor

1. Ir al portal → Gestión de Proveedores → Crear Proveedor  
   O ejecutar: `python create_tenants.py --create` (agregar a la lista `NEW_TENANTS` primero)
2. Ejecutar diagnóstico: `python send_invitations.py`
3. Obtener ID_TOKEN en netoinsight.soyneto.com → F12 → Network → embed-url → Authorization header
4. Enviar: `export ID_TOKEN="eyJ..." && python send_invitations.py --send`

> ⚠️ El `proveedorIdInterno` debe coincidir **exactamente** con el valor del filtro en Tableau Cloud.
